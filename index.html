Phase 1 — Quick TCP Scan

PHASE 1 — QUICK ALL-PORT TCP SCAN

Run this FIRST on all targets simultaneously.

NMAP (reliable)
nmap -p- --min-rate 5000 --open -T4 $TARGET -oN ~/exam/recon/$TARGET-allports.txt
RUSTSCAN (fastest)
rustscan -a $TARGET --ulimit 5000 -- -sC -sV -Pn -oN nmap_full

EXTRACT PORTS

ports=$(grep -oP '\d+(?=/open)' ~/exam/recon/$TARGET-allports.txt | tr '\n' ',' | sed 's/,$//')
echo "Open ports: $ports"

NOTE: Run nmap on all 3 standalone IPs in background tabs:

nmap -p- --min-rate 5000 -T4 10.10.10.1 -oN recon/ip1.txt &
nmap -p- --min-rate 5000 -T4 10.10.10.2 -oN recon/ip2.txt &
nmap -p- --min-rate 5000 -T4 10.10.10.3 -oN recon/ip3.txt &

Phase 2 — Targeted Version + Scripts

PHASE 2 — TARGETED VERSION + SCRIPT SCAN

Run after extracting open ports from Phase 1.

FULL TARGETED SCAN

nmap -sC -sV -p$ports $TARGET -oN ~/exam/recon/$TARGET-targeted.txt -oX ~/exam/recon/$TARGET-targeted.xml

SPECIFIC SCRIPT CATEGORIES

nmap --script "vuln" -p$ports $TARGET -oN ~/exam/recon/$TARGET-vulns.txt
nmap --script "default,safe" -p$ports $TARGET

SMB (if port 139/445 open)

nmap -p139,445 --script smb-vuln*,smb-enum-shares,smb-enum-users,smb-os-discovery $TARGET
nmap -p139,445 --script smb-security-mode $TARGET

HTTP (if port 80/443/8080 open)

nmap -p80,443,8080 --script http-methods,http-title,http-robots.txt,http-headers $TARGET

EXAMINE OUTPUT

Read targeted.txt carefully:

- Note ALL service versions exactly (searchsploit them)

- Note unusual ports

- Note OS from TTL or smb-os-discovery

- Note hostnames → add to /etc/hosts

Phase 3 — UDP + OS Detection

PHASE 3 — UDP + OS DETECTION

UDP TOP 20 SCAN

sudo nmap -sU --top-ports 20 -T4 $TARGET -oN ~/exam/recon/$TARGET-udp.txt

CRITICAL UDP PORTS TO CHECK

UDP 53 → DNS zone transfer attempt

UDP 69 → TFTP (often misconfigured, anonymous read/write)

UDP 111 → RPCbind → NFS enumeration

UDP 161 → SNMP → community string bruteforce → huge info dump

UDP 500 → IKE/IPsec

UDP 1194 → OpenVPN

SNMP (if open)

snmpwalk -c public -v2c $TARGET 1.3.6.1.2.1.25.4.2.1.2  # Windows processes

OS DETECTION

sudo nmap -O $TARGET

# TTL 128 = Windows
# TTL 255 = Network device

DNS (port 53 open)

dig axfr @$TARGET $DOMAIN # zone transfer
